USAGE:

You need first a tracking ID with a WEB profile.

1. Unzip the gawebwrapper.js.zip file and place the file (gawebwrapper.js) next to the html file generated by unity at the root level.

2. Add the following line in the header of the unity3D generated html file.

<script type="text/javascript" src="gawebwrapper.js"></script>


Comments :

The gawebwrapper uses the new analytics.js for universal app on Google Analytics (https://developers.google.com/analytics/devguides/collection/analyticsjs/?hl=fr)
